Related Questions

How do I add a new task?
You can add a new task by entering the task description in the "Task description" field and the assignee's name in the "Assign to" field. Click the "Add Task" button to add the task.
How can I assign a task to someone?
To assign a task to someone, enter the task description in the "Task description" field and the assignee's name in the "Assign to" field. Click the "Add Task" button to assign the task.
Can I edit or delete a task after adding it?
Yes, you can edit or delete a task after adding it. Click on the veiw all task, you want to edit or delete, and then click the respective action button within the task box.
How do I keep track of assigned tasks?
You can keep track of assigned tasks by viewing the list of tasks. Each task displays its description and assigned person. To remove a task, click the "Remove" button within the task box.
